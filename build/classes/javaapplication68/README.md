# JavaApplication68 - Gestor de Tareas

## Descripción

Aplicación Java desarrollada en NetBeans que implementa un sistema básico de gestión de tareas. Permite representar y organizar tareas con su descripción y estado de completitud.

## Estructura del proyecto

| Clase | Descripción |
|-------|-------------|
| `JavaApplication68` | Clase principal con el método `main`, punto de entrada de la aplicación. |
| `Tarea` | Modelo que representa una tarea, con atributos `descripcion` (String) y `completada` (boolean). |
| `GestorTareas` | Gestor que mantiene una lista de objetos `Tarea` mediante un `ArrayList`. |

## Requisitos

- Java JDK 8 o superior
- NetBeans IDE (recomendado)

## Ejecución

1. Abre el proyecto en NetBeans.
2. Compila el proyecto con **Build Project** (`F11`).
3. Ejecuta con **Run Project** (`F6`).
